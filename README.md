# Базовый код

## Модели данных (Modal)

## IProduct и Product

### IProduct

Интерфейс, описывающий продукт.

| Поле          | Тип              | Описание                          |
| ------------- | ---------------- | --------------------------------- |
| `id`          | `string`         | Уникальный идентификатор продукта |
| `description` | `string`         | Описание продукта                 |
| `image`       | `string`         | URL изображения продукта          |
| `title`       | `string`         | Название продукта                 |
| `category`    | `string`         | Категория продукта                |
| `price`       | `number \| null` | Цена продукта, может быть `null`  |

### Product

Класс, реализующий интерфейс продукта.

```typescript
class Product implements IProduct {
  constructor(
    public id: string,
    public description: string,
    public image: string,
    public title: string,
    public category: string,
    public price: number | null
  ) {}
}
```


## IBasket и Basket
# IBasket
Интерфейс, описывающий корзину с продуктами.

| Поле    | Тип          | Описание                   |
| ------- | ------------ | -------------------------- |
| `items` | `IProduct[]` | Массив продуктов в корзине |

# Методы:

- `add(product: IProduct)`: void - Добавляет продукт в корзину.
- `remove(id: string)`: void - Удаляет продукт из корзины по идентификатору.
# Basket
Класс, реализующий интерфейс корзины с продуктами.

```typescript
class Basket implements IBasket {
  items: IProduct[] = [];

  add(product: IProduct) {
    this.items.push(product);
  }

  remove(id: string) {
    this.items = this.items.filter(item => item.id !== id);
  }
}
```

## Отображение данных (View)

### Импортируемые модели
Модуль импортирует несколько интерфейсов и классов из модуля models, таких как IProduct, Product, IBasket, Basket, IOrder, Order, IUser и User.

### Основные функции слоя отображения
### updateBasketCounter
- Описание: Обновляет счетчик товаров в корзине, отображаемый в элементе .header__basket-counter.
- Параметры:
`count` - Количество товаров в корзине.
### displayProducts
- Описание: Отображает список продуктов в галерее.
- Параметры:
`products` - Массив продуктов для отображения.
### showProductDetails
- Описание: Показывает детали продукта в модальном окне.
- Параметры:
`product` - Продукт, детали которого нужно показать.
### addToBasket
- Описание: Добавляет продукт в корзину.
- Параметры:
`product` - Продукт для добавления в корзину.
### updateBasketView
- Описание: Обновляет список продуктов в корзине.
- Параметры:
`basket` - Объект корзины.
### removeFromBasket
- Описание: Удаляет продукт из корзины.
- Параметры:
`productId` - Идентификатор продукта для удаления.
### initOrderForm
- Описание: Инициализирует форму заказа.
- Параметры:
`order` - Объект заказа.
### initContactForm
- Описание: Инициализирует форму контактной информации.
- Параметры:
`user` - Объект пользователя.
### showOrderSuccess
- Описание: Показывает сообщение об успешном оформлении заказа.

## Пример использования функций
 1. Обновление счетчика корзины:

- Функция `updateBasketCounter` используется для обновления отображаемого количества товаров в корзине при добавлении или удалении продуктов.
 
 2. Отображение продуктов:

- Функция `displayProducts` отображает продукты в галерее, используя шаблон карточки продукта.

 3. Просмотр деталей продукта:

- Функция `showProductDetails` открывает модальное окно с деталями выбранного продукта.

 4. Добавление продукта в корзину:

- Функция `addToBasket` добавляет выбранный продукт в корзину и обновляет счетчик товаров.

 5. Обновление вида корзины:

- Функция `updateBasketView` обновляет список продуктов в корзине и общую стоимость корзины.

 6. Удаление продукта из корзины:

- Функция `removeFromBasket` удаляет указанный продукт из корзины и обновляет вид корзины и счетчик товаров.

 7. Инициализация формы заказа:

- Функция `initOrderForm` настраивает форму заказа для обработки и отправки данных заказа.

 8. Инициализация формы контактной информации:

- Функция `initContactForm` настраивает форму контактной информации для обработки и отправки данных пользователя.

 9. Сообщение об успешном заказе:

- Функция `showOrderSuccess` показывает сообщение об успешном оформлении заказа в модальном окне.

## Описание функций
**`updateBasketCounter`**
Эта функция обновляет счетчик товаров в корзине. Она получает количество товаров и обновляет элемент, отображающий это количество.

**`displayProducts`**
Эта функция отображает список продуктов в галерее. Она получает массив продуктов и создает карточки продуктов, используя шаблон, и добавляет их в галерею.

**`showProductDetails`**
Эта функция показывает детали продукта в модальном окне. Она получает продукт и обновляет модальное окно с деталями этого продукта, используя шаблон.

**`addToBasket`**
Эта функция добавляет продукт в корзину. Она получает продукт, добавляет его в корзину и обновляет счетчик товаров.

**`updateBasketView`**
Эта функция обновляет список продуктов в корзине. Она получает объект корзины и обновляет вид корзины, отображая продукты и общую стоимость.

**`removeFromBasket`**
Эта функция удаляет продукт из корзины. Она получает идентификатор продукта, удаляет продукт из корзины и обновляет вид корзины и счетчик товаров.

**`initOrderForm`**
Эта функция инициализирует форму заказа. Она получает объект заказа и настраивает форму для обработки и отправки данных заказа.

**`initContactForm`**
Эта функция инициализирует форму контактной информации. Она получает объект пользователя и настраивает форму для обработки и отправки данных пользователя.

**`showOrderSuccess`**
Эта функция показывает сообщение об успешном оформлении заказа. Она обновляет модальное окно с сообщением об успешном заказе, используя шаблон.
